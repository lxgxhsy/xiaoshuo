# 角色档案

## 许观潮

- 身份：潮钟署低级校时档案员。
- 核心欲望：证明哥哥当年不是白昼误差的肇事者。
- 长期目标：查清全城缺失时间的真实来源。
- 当前目标：解释海晷港西码头出现的七分钟空白。
- 资源：档案调阅权限、老式怀表、一枚欠债未还的人情。
- 已知信息：钟塔记录可以被篡改，但篡改会留下盐雾形态的误差。
- 认知层：2。
- 力量层级：0。
- 约束：没有正式刻律资格，不能直接接触高危材料。

## 夏葵

- 身份：蓝蜡商会走私联络人。
- 核心欲望：让患回声化的母亲恢复稳定。
- 长期目标：获得足够无声蜡和旧钟屑，买断母亲的治疗契约。
- 当前目标：把一面盐镜交给未知买家。
- 资源：盐镜、地下航道、人脉、假身份。
- 已知信息：盐镜里保存着一个被删除的证词。
- 认知层：2。
- 力量层级：1。
- 约束：欠商会三次不可拒绝委托。

## 梁烛

- 身份：潮钟署外勤督察。
- 核心欲望：维持城市秩序，阻止异常时间扩散。
- 长期目标：升入内署，取得白昼误差档案。
- 当前目标：封锁西码头七分钟空白事件。
- 资源：外勤队、封锁令、三枚无声蜡、审讯权限。
- 已知信息：西码头事件与无声修会有关，但证据不足。
- 认知层：3。
- 力量层级：2。
- 约束：不能让公众知道潮钟署也无法解释钟塔误差。

## 裴穆

- 身份：失明钟表匠，疑似无声修会成员。
- 核心欲望：让城市停止被错误时间统治。
- 长期目标：破坏七座公钟的同步机制。
- 当前目标：找回被夏葵带走的盐镜。
- 资源：盲听律印、旧钟工坊、两个沉默誓约者。
- 已知信息：盐镜保存的证词会指向第十一次钟鸣。
- 认知层：4。
- 力量层级：3。
- 约束：听见真实钟声时会短暂失去自我。
