# 第一卷：七分钟空白

## 卷目标

第一卷让读者理解海晷港的基本秩序、律印系统的代价、信息危险和主角无豁免原则。结尾揭示西码头事件不是孤立异常，而是白昼误差的回声。

## 章节草案

1. 第一个缺失的七分钟：许观潮发现西码头钟表记录全部短缺七分钟。
2. 盐镜里的证词：夏葵使用永动摆轮穿过水闸，失去七个未来分钟，盐镜证词开始互相矛盾。
3. 督察的封锁线：梁烛判断事件危险等级，消耗封锁令压住记录，并识别永动摆轮的真实成本。
4. 盲钟匠：裴穆追索盐镜，展现高层信息会伤害低层理解者。
5. 无声蜡：许观潮为取得证据燃烧无声蜡，失去母亲最后一句话的声音记忆。
6. 三份不一致的口供：同一事件从许观潮、夏葵、梁烛三个视角产生矛盾版本。
7. 第八座钟：众人发现城市档案里曾短暂存在过一座被删除的钟塔。
8. 代价校时：许观潮没有赢过裴穆，只用证据迫使梁烛介入，代价是哥哥旧案被重新标为高危。

## 第一卷硬规则兑现点

- 危险信息：盐镜证词会腐蚀记忆。
- 资源消耗：无声蜡只能使用一次，使用者失去听觉记忆。
- 越级冲突：许观潮不能正面击败裴穆，只能利用证据、封锁令和组织目标逃脱。
- 多视角矛盾：三份口供都自洽，但没有任何一份给出完整真相。
- 不可知层：第十一次钟鸣只通过缺失时间和互相矛盾的钟表记录出现。
