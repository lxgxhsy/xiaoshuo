# 技能与装置设定

## 永动摆轮

- 类型：第七律阶装置 / 高危技能。
- 黑市称呼：永动机。
- 真实机制：把显性燃料消耗转移为时间债。
- 激活条件：潮汐铜闭环、盐镜校准、至少一名刻律者提供本名锚。
- 可见效果：目标机器或仪式在无燃料输入下继续运转。
- 必要成本：未来分钟、旁观者时间债或本名锚。
- 副作用：时间缺帧、听觉延迟、回声化、身份松动。
- 硬规则：永动摆轮不能创造免费能量，只能移动成本。

## BDD 约束

- 使用永动摆轮必须记录 `used_skills = ["perpetual_pendulum"]`。
- 事件必须记录 `consumed_resources`。
- 若角色不知道成本，事件仍要在客观图谱中记录隐藏成本。
- 若正文表现为“没有任何代价”，验证器必须失败。
- 若主角使用该装置，也必须承受同等成本。
