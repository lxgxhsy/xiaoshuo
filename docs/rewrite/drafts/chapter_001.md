# 第一章 第一个缺失的七分钟

西码头的雾在凌晨四点二十三分贴上玻璃，像一层没有干透的盐。

许观潮把第六本钟簿翻到末页，指腹停在同一个空格上。货船入港、缆车换班、巡灯人报点、税关盖章，所有记录都严丝合缝地少了七分钟。不是一座钟慢了，也不是某个书记员偷懒，而是整片码头在纸面上被人剪掉了一小段。

他没有立刻写报告。

潮钟署的档案室有一条不成文的规矩：凡是能用人祸解释的异常，都先按人祸登记。这样外勤才有嫌疑人可以抓，署长才有理由向议会要经费，普通市民也能继续相信七座公钟永远准确。

可许观潮知道，钟簿不会一起撒谎。墨水能改，签名能仿，纸页能替换，唯独盐雾留下的误差改不了。每当时间记录被异常力量动过，纸边会浮出细得像鱼骨的白线。

这一次，鱼骨线从末页一直爬进封皮。

他从抽屉里取出半截无声蜡。蜡身呈灰蓝色，握在手里没有温度，像一块被海水泡过的骨。外勤督察梁烛曾经说过，没有执照的人不该点它；档案室主任也说过，点燃无声蜡之后，别急着抱怨自己丢了什么声音。

许观潮把蜡立在铜碟中央，划亮火柴。

火苗落下的一瞬间，整间档案室安静得不像现实。窗外缆车的铁轮声消失了，远处潮钟的余震消失了，连他自己的呼吸都像被一只手按进水里。

钟簿上的白线却亮了起来。

它们不是线，而是一排被压扁的字：第十一次报点之后，证人夏葵离开西码头。

许观潮盯着“第十一次”四个字，眼睛开始刺痛。他知道七座公钟每晚只报十次夜点，多出来的一次不该存在，也不该被低级档案员读见。无声蜡燃到一半时，他听见记忆深处有什么东西被擦掉了。

那是母亲临终前留给他的最后一句话。

他还记得病床、白帘、药水味和哥哥站在门口的影子，却再也想不起她的声音。硬规则生效得干净利落，没有因为他姓许、没有因为他查的是哥哥旧案，就多留一分情面。

档案室门外传来三下敲击。声音重新回到世界，粗糙得像砂纸。

“开门。”梁烛在外面说，“西码头封锁了。你最好还没把不该看的东西写进报告。”

许观潮合上钟簿，把剩下的无声蜡灰收进纸袋。他没有胜利，也没有获得什么可靠答案。他只有一个名字、一句不该存在的报点，和一段已经付出去的声音。

门锁转动时，他在报告纸第一行写下：西码头事件，暂不按人祸归档。
